# Tinkoff Mobile Redesign

Практика Auto Layout и сложных карточек в Figma.

## Задача

Воссоздать главный экран Tinkoff с нуля, используя современные возможности Figma.

## Что было сделано

- Созданы сложные карточки с Auto Layout
- Настроена поисковая строка и приветствие
- Реализованы кнопки быстрых действий
- Сделана нижняя навигация
- Всё собрано в один responsive экран
